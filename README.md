## Running Locally
* Clone repo
* Run ``npm install``
* Create a ``Config.ts`` file
    * Use ``Config-template.ts`` to fill in you OAuth credentials
    * Rename ``Config-template.ts`` to ``Config.ts``
* Run ``npm run development`` to boot up your üçûbot
